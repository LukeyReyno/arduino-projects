# RGB LED Tuner
Simple RGB LED Tuner to get specific colors from a potentiometer.

Source Code Included

## Materials Used:
* Arduino Uno
* 7 Resistors (Specific Ratings not Necessary)
* 1 Push Button
* 1 10k Potentiometer
* 1 RGB LED
* (Optional 3 LEDS: Red, Green, and Blue)
* Male to Male Wires
